Optiplex 7020 sff

My initial homelab setup started with a Dell Optiplex 7020 sff machine that came with an Intel Core i5 4590 Quad-Core CPU with a base clock of 3.3GHz and boost clock of 3.7 GHz, 8Gb of DDR3-1600 RAM, and a 500Gb HDD. I swapped the DVD drive out with a 2.5″ SATA HDD expansion bay and installed a 60Gb laptop HDD into it to use as a boot drive (This was later upgraded to a 128Gb SSD). The next upgrade to the machine was another 8Gb of RAM and then a larger HDD. This machine is currently used to host multiple virtual machines that are used for other projects with my lab.

The machine runs Proxmox VE version 6.3-2 and hosts a Plex server, a Minecraft server, a Jenkins lab and a Jira lab at this time.

I installed Plex onto a virtual machine that lives on the Optiplex 7020, running Ubuntu Server version 20.04-3, to be able to stream movies from my storage server. This went very smoothly, as I didn’t have to worry about any hardware issues and I was able to find some pretty handy Plex installation guides. This installation still works without issue and is used often for watching movies in different parts of the home.

I, of course, had to create a Minecraft server for my home as well. This was installed onto another virtual machine on my Optiplex 7020 that also runs Ubuntu Server version 20.04-3. This went smoothly, as it wasn’t too hard to find an installation guide online that held the right commands for downloading the Java files needed and installing the game server itself onto the machine. This server is used for casual Minecraft games with any guests who want to play, and it is constantly evolving as I tinker with installing mod packs and learning how Minecraft server administration is done.

After the successful installation of my Minecraft server, I decided to start experimenting with some tools that I see used in the DevOps field, such as Jenkins and Jira. Both of these tools were installed onto their own virtual machines running, yet again, Ubuntu Server version 20.04-3 (handy little operating system).

The first of the two machines that I got up and running was the Jenkins server. This installation completed without any hiccups and it still runs in my home for me to tinker with.

The second of the two machines was the Jira one. This installation had no issues either, and I have not yet tinkered with it at the time of this writing.
