“Big Box” server

This server is essentially the “guts” of my home lab. It currently contains 2 x Intel Xeon e5-2680 V3 CPUs, each with 12 cores running at 2.5 Ghz base frequency and 3.3Ghz turbo frequency. It uses DDR4 Ram and contains about 12 hard drives of varying capacities.

I have Proxmox Virtual Environment installed on this server and it is used for the grand majority of the tinkering I do with various services and operating systems.

I currently have several virtual machines running on it, one of which is an Ubuntu Server installation used to host my Grafana instance to monitor all of the other systems I use. I use just one dashboard currently to monitor system CPU temps and frequencies as well as disk and network usage for all of the physical machines I have.

There are several other virtual machines running on this server with various versions of Windows installed on them, all joined to a Windows Server 2022 domain so I can experiment with adjusting user permissions and running an IIS website on the Windows Server installation.

All of my virtual machines are backed up to a local drive within the server, which is them mirrored to another drive on my TrueNas server (Dell R720) for redundancy in case of a drive corruption or any other weirdness that may occur.
