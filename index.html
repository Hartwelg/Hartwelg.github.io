About

My name is Gauge Hartwell and I will be completing a degree in Cybersecurity from Oregon State University in June of 2021.

I have experience with the C, C++, and Python languages as well as the Flask and Scapy libraries for Python. I gained experience with Flask while learning Cloud Development and Scapy while learning about network security. My experience in C comes from learning about systems programming and networking.

Interests include: Video Games, Woodwork, 3D Printing, Astronomy, Raspberry Pi (and other SBCs), and Arduino.

Some notable personal projects:

  • Arduino lamp control.
  • 3D models of past woodwork projects (shown on this website).
  • PiHole on my home network to assist in blocking ads and trackers while on the internet.
  • Raspberry Pi home server to host this website (locally).
  • Python password strength checker.
  • Python implementation of OTP.

Future planned personal projects:

  • Website built in Flask and interacting with a MariaDB database with PHP (exercise in database and website security).
  • Raspberry Pi 3D printer upgrade to allow for wifi printing and print monitoring.
  • Raspberry Pi/Arduino feature tracking for a telescope.
