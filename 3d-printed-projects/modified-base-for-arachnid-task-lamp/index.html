The original base for the “Arachnid Task Lamp” project on Thingiverse was not to my liking, so I designed this one to replace it. It uses the same ball joint to mount the rest of the lamp on, but the base uses more basic shapes and takes much less time to print. It supports the weight of four flashlights just as well as the original base does.
